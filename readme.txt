OxyProps Lite makes Open Props CSS variables available in your WordPress environment.

== Description ==

OxyProps Lite is an Open Source WordPress plugin making the Open Props CSS Variables collection easily available in your WordPress environment.

* View [OxyProps Lite source code on Github](https://github.com/thewebforge/oxyprops-lite).
* View [Open Props source code on Github](https://github.com/argyleink/open-props).

OxyProps Lite makes all 351 Open Props design tokens availble in your WordPress environment so you can use theme wherever you can type custom CSS or input properties values.

PropsPacks categories includes :
* Colors
* Gradients
* Shadows
* Aspect Ratios
* Typography
* Easing
* Animations
* Sizes
* Borders
* Z-Index
* Media Queries

From OxyProps Lite dashboard, you can choose to load the full bundle, or to select individual PropsPacks to pick only the ones necessary for your project.

OxyProps Lite allows you either to enqueue stylesheets with <link> tags or to inline CSS if you prefer saving http requests.

== FAQ ==

= What is OxyProps Lite? =

OxyProps Lite is an Open Source WordPress plugin making the Open Props CSS Variables collection easily available in your WordPress environment.

= What is Open Props? =

Open Props is an Open Source collection of CSS Design Tokens (elementary design bricks) in the form of CSS Custom Properties (aka CSS Variables).

= Will OxyProps Lite work with X/Y/Z (you name it) site builder? =

Yes, OxyProps Lite is pure CSS and will work wherever you can type CSS values in a text field. This can be custom CSS fields in your WordPress Customizer, Properties value fields in your site buidler, or custom stylesheets in your site builder.

= What is OxyProps Pro? =

OxyProps Pro is a Premium WordPress plugin including an extended version of Open Props, a large (>6000) CSS Utility Classes library, Add-ons for Oxygen Builder (context menu for easy acces to the framework), and Oxygen Custom Elements. A Bricks builder version is currently under development.
